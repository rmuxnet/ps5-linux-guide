# What Is Xenia-Edge?

Xenia Edge is yet another experimental fork of the Xenia emulator, originally based on Xenia Canary. The focus is on faster iteration, higher default game compatibility, usability and platform support.

# 1.Auto Installation

The guide will provide a automated installation script for xenia edge:

```bash
curl -sSL https://raw.githubusercontent.com/rmuxnet/ps5-linux-guide/main/docs/public/scripts/Xenia/xenia.sh | bash
```

Due to the intricate nature of configuring the Xenia Edge emulation layer, users are strongly encouraged to thoroughly review the official documentation framework prior to initialization:
