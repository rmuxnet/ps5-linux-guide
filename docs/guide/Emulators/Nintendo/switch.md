# Switch Emulation 

Due to legal enforcement actions, official distribution channels and source repositories for both the Yuzu and Ryujinx Nintendo Switch emulation projects have been permanently decommissioned. Consequently, obtaining verified, uncompromised binaries from upstream sources is no longer possible.

To ensure stability and system security, it is highly recommended to manage Nintendo Switch emulation deployment via the EmuDeck framework. The EmuDeck ecosystem maintains pre-configured, vetted iterations of these emulation cores, mitigating the risks associated with downloading third-party binaries from untrusted external mirrors.

> [!WARNING]
> Users should exercise extreme caution when sourcing individual standalone binaries online. Utilizing the automated EmuDeck installation pipeline remains the safest and most reliable method to integrate these runtime environments.
